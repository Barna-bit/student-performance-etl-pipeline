# Student Performance ETL Pipeline

## Project Overview
Built an end-to-end ETL pipeline using the Kaggle Student Performance dataset to extract, transform, and analyze student performance data.

## Technologies Used
- Python
- Pandas
- SQLAlchemy
- Jupyter Notebook

## ETL Process
### Extract
- Loaded raw student performance data from Kaggle CSV.

### Transform
- Removed duplicate records.
- Standardized column names.
- Created average score, performance category, and result status.

### Load
- Exported the transformed dataset for reporting and analysis.

## Key Insights
- Average scores by gender.
- Impact of test preparation courses.
- Performance distribution.
- Top-performing students.

## Dataset
Students Performance in Exams Dataset (Kaggle)
